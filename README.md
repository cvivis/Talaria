## 🎉 프로젝트 소개

💙 <b>서비스 설명</b>

> API들을 세 가지 역할로 나누어 사용하는 서비스입니다.\
> `사용자`들은 API들을 신청하여 사용할 수 있습니다.\
> `개발자`들은 API들을 만들어 탈라리아에 등록할 수 있습니다.\
> `관리자`들은 계정과 API들을 관리합니다.
